## :department_store: Floating-Population Analyzing Service
* 쇼핑몰 유동인구 분석 서비스
  * 유동인구 수를 측정하여 가장 최적화된 매장배치 및 매장 내 상품배치 전략을 파악하고, 이를 통해 쇼핑몰의 이윤을 최대화할 수 있도록 한다.

  * 쇼핑몰 내의 장소별, 시간별 유동인구 수를 측정하고 변화량을 분석하여 막대그래프, 히트맵 등의 시각적인 정보로 제공한다.
